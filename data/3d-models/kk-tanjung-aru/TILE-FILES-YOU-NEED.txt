HOW TO USE THIS LIST
--------------------
1. Open your full 3D Tiles export (the folder that contains "top" with Level_11
   through Level_16).
2. Copy the "top" folder into:  data/3d-models/kk-tanjung-aru/
   So you get:  data/3d-models/kk-tanjung-aru/top/Level_11/ ... Level_16/
3. Or copy only the files listed above from each Level_* folder into the
   matching Level_* folder here (create Level_11 ... Level_16 if needed).
4. File names must match exactly (including + and _ and zeros).

After all 84 files are in place, the 3D model will load when you open the
viewer for Tanjung Aru.
